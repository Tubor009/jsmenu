JS Menu

Overview

This lib allows to create a classic drop-down menu. There are five classes:

  • JSMenuBar
  • JSMenu
  • JSMenuItem
  • JSRadioButton
  • JSCheckBox

JSMenuBar

Constructors
JSMenuBar()

Constructs a new JSMenuBar with default options.

JSMenuBar(options:Object)

Constructs a new JSMenuBar with options.

options is a hash with objects that describe the style of the menu:

  • menuBar - styles for the menu bar container.
  • rootMenuOut - styles for each root menu when mouse outside of them.
  • rootMenuOver - styles for each root menu when mouse inside of them.
  • subMenu - styles for submenu container.
  • itemMenuOut - styles for each item in the submenus when mouse outside item.
  • itemMenuOver - styles for each item in the submenus when mouse inside item.

default options:

Show 
{
    menuBar:{
        width:"100%",
        background:"#d3d3d3"
    },
    rootMenuOut:{
        fontFamily:"georgia",
        padding:"3px 15px",

        textAlign:"left"
    },
    rootMenuOver:{
        fontFamily:"georgia",

        cursor:"pointer"

    subMenu:{
        border:"1px solid black"
    },
    itemMenuOut:{
        fontFamily:"georgia",
        background:"#d3d3d3",
        textAlign:"left",
        padding:"3px 20px"
    },
    itemMenuOver:{
        fontFamily:"georgia",
        background:"#a9a9a9",
        cursor:"pointer"
    }
}

Methods
add(menu:JSMenu)

Adds JSMenu instance to the end of the menu bar.

appendTo(elem:Element)

Adds JSMenuBar instance to the specified element.

destruct()

Deletes all functionality and removes menu from page;

build()

Creates handlers and builds all items what are added before this calling.
Note, call this method when DOM-structure is loaded. If you do not call this method, the menu will be constructed at the first mouseover on menubar.

JSMenu

Constructors
JSMenu(label:String)

Constructs a new JSMenu with specified label;

JSMenu(label:String, iconUrl:String)

Constructs a new JSMenu with specified label and icon. Icon located in left side of menu; So you need be careful with "padding" - style of this menu.

Methods
add(item:JSMenuItem)

Adds JSMenuItem instance to the end of this submenu.

add(item:JSRadioButton)

Adds JSRadioButton instance to the end of this submenu.

add(item:JSCheckBox)

Adds JSCheckBox instance to the end of this submenu.

addSeparator(color:String)

Adds a separator line with the specified color at the end of this menu. The default color - black.

addIcon(url:String)

Adds icon to the left side of this menu.

JSMenuItem

Constructors
JSMenuItem(label:String, options:Object)

Constructs a new JSMenuItem with specified label;

options(optional):
  • icon - url of image. Icon will be on the left side of item.
  • clickHandler - callback function.

Methods
addClickHandler(f:Function)

Adds callback function

addIcon(url:String)

Adds icon. Icon will be on the left side of item.

JSRadioButton

Constructors
JSRadioButton(label:String, options:Object)

Constructs a new JSRadioButton with specified label;

options(optional):
  • checked - boolean state .
  • clickHandler - callback function.

Methods
addClickHandler(f:Function)

Adds callback function

getState():Boolean

Returns state of this item.

setState(b:Boolean)

Sets state for this item.

JSCheckBox

Constructors
JSCheckBox(label:String, options:Object)

Constructs a new JSCheckBox with specified label;

options(optional):
  • checked - boolean state .
  • clickHandler - callback function.

Methods
addClickHandler(f:Function)

Adds callback function

getState():Boolean

Returns state of this item.

setState(b:Boolean)

Sets state for this item.

Examples

Example 1 Show code

        var menuContainer = document.getElementById("menu");

        var jsMenuBar = new JSMenuBar({
            menuBar:{
                width:"100%"
            },
            itemMenuOut:{
                fontSize:"12px"
            }
        });

        jsMenuBar.appendTo(menuContainer);

        var menuFile = createMenuFile();
        jsMenuBar.add(menuFile);

        jsMenuBar.add(createMenuEdit());

        jsMenuBar.add(createMenuView());

        jsMenuBar.add(createMenuService());

            function createMenuFile(){
                  var menuFile = new JSMenu("File");
                  var itemNew = new JSMenuItem("new...");
                  menuFile.add(itemNew);
                  itemNew.addClickHandler(function(e){
                      alert("New...");
                  })

                  var itemOpen = new JSMenuItem("Open...",{
                     icon:"docImgs/open.png",
                     clickHandler:function(e){
                         alert("Open");
                     }
                  });
                  menuFile.add(itemOpen);

                  menuFile.add(new JSMenuItem("Save As...",{
                     clickHandler:function(e){alert("Save As...")},
                     icon:"docImgs/save_blue.png"
                  }));

                  menuFile.add(new JSMenuItem("Save All",{
                     clickHandler:function(e){alert("Save All")},
                     icon:"docImgs/saveall_blue.png"
                  }));
                  menuFile.addSeparator();
                  menuFile.add(new JSMenuItem("Exit",{
                     clickHandler:function(e){alert("Exit")}
                  }));

                  return menuFile;
            }

            function createMenuEdit(){
                var menuEdit = new JSMenu("Edit");
                menuEdit.add(new JSMenuItem("Undo"));
                menuEdit.add(new JSMenuItem("Redo"));
                menuEdit.addSeparator();
                menuEdit.add(new JSMenuItem("Copy"));
                menuEdit.add(new JSMenuItem("Cut"));
                menuEdit.add(new JSMenuItem("Paste"));
                menuEdit.add(new JSMenuItem("Delete"));
                menuEdit.addSeparator();
                menuEdit.add(new JSMenuItem("Find",{
                    icon:"docImgs/find.png"
                }));
                menuEdit.add(new JSMenuItem("Find And Replace...",{
                    icon:"docImgs/find_all.png"
                }));
                return menuEdit;
            }

            function createMenuView(){
                var viewMenu = new JSMenu("View");
                var checkedStatusBar = new JSCheckBox("Status Bar",{
                    checked:true,
                    clickHandler:function(e){
                          alert(checkedStatusBar.getState());
                    }
                });
                viewMenu.add(checkedStatusBar);
                viewMenu.add(new JSCheckBox("Toolbar"));
                viewMenu.add(new JSCheckBox("Navigation Bar"));
                viewMenu.add(new JSCheckBox("Show gadgets"));
                viewMenu.addSeparator();

                var menuScale = new JSMenu("Scale");
                viewMenu.add(menuScale);

                var radio1 = new JSRadioButton("100%",{
                    checked:true,
                    clickHandler:function(e){
                        radio1.setState(true);
                        radio2.setState(false);
                    }
                });

                var radio2 = new JSRadioButton("50%",{
                    checked:false,
                    clickHandler:function(e){
                        radio2.setState(true);
                        radio1.setState(false);
                    }
                });
                menuScale.add(radio1);
                menuScale.add(radio2);

                var menuPageDisplay = new JSMenu("Page Display");

                menuPageDisplay.add(new JSCheckBox("Two Page View",{checked:true}));
                menuPageDisplay.add(new JSCheckBox("Enable Scrolling"));
                menuPageDisplay.addSeparator();
                menuPageDisplay.add(new JSCheckBox("Automatically Scrolling"));

                viewMenu.add(menuPageDisplay);

                return viewMenu;
            }

            function createMenuService(){
                var menuService = new JSMenu('Service');

                var menuLanguage= new JSMenu("Language");

                var radio1 = new JSRadioButton("Automatically",{
                    checked:true,
                    clickHandler:function(e){
                        radio1.setState(true);
                        radio2.setState(false);
                        radio3.setState(false);
                    }
                });

                var radio2 = new JSRadioButton("English",{
                    checked:false,
                    clickHandler:function(e){
                        radio2.setState(true);
                        radio1.setState(false);
                        radio3.setState(false);
                    }
                });

                var radio3 = new JSRadioButton("Russian",{
                    checked:false,
                    clickHandler:function(e){
                        radio3.setState(true);
                        radio1.setState(false);
                        radio2.setState(false);
                    }
                });

                menuLanguage.add(radio1);
                menuLanguage.addSeparator();
                menuLanguage.add(radio2);
                menuLanguage.add(radio3);
                menuService.add(menuLanguage);


                var menuSetting = new JSMenu("Settings","docImgs/settings.png");
                menuService.add(menuSetting);
                menuSetting.add(new JSMenuItem("Template Settings..."));
                menuSetting.add(new JSMenuItem("Encode Settings..."));
                menuSetting.add(new JSMenuItem("Font Settings..."));

                var menuAppSetting = new JSMenu("Application settings");
                menuAppSetting.add(new JSMenuItem("Editor Settings..."))

                var menuDB = new JSMenu("Data Base Settings");
                menuDB.add(new JSMenuItem("Open Current Base"));
                menuDB.addSeparator();
                menuDB.add(new JSMenuItem("Access Settings",{
                    icon::"docImgs/access.png"
                }));
                menuDB.add(new JSMenuItem("Choose Provider"));
                menuAppSetting.add(menuDB);
                menuAppSetting.add(new JSMenuItem("Notifications..."));
                menuAppSetting.add(new JSMenuItem("Plugins..."));

                menuSetting.add(menuAppSetting);
                return menuService;
            }
